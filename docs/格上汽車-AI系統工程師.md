## 工作內容

負責開發與維護 AI 應用產品：

1. 負責 LLM（大型語言模型）驅動之 **AI Agent** 系統的架構設計與核心模組開發，包含任務規劃、記憶管理、工具調用與流程執行
2. 利用 **Google ADK / Vertex AI** 等工具，構建具備自主推理能力的 Agent 系統，處理高複雜度的商業決策流程
3. 深度優化 **RAG（檢索增強生成）** 架構，解決模型幻覺問題，確保在垂直領域知識下的回覆精準度
4. 應用 RAG、Skills、**Function Calling** 等關鍵技術，打造具備上下文理解與知識注入能力的智慧系統
5. 建立 Agent 行為的**評估機制與自動化測試工具**，持續優化任務完成率、回覆品質與系統可靠度
6. 推動 AI Agent 在垂直場景中的應用（搜尋、交易、深度分析與自動化數據工程），釋放大型語言模型的最大價值
